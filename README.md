# learnUnderscore

underscore不仅是一个很好用的JS库，更是一个值得仔细学习和品读的JS学习范本，里面不仅有很多对象、数组的处理方式，还有很多值得自己借鉴和学习的处理方式。

本仓库并不是教大家如何使用underscore，而是在自己学习underscore源码过程中总结出的一些JS技巧和经验。
